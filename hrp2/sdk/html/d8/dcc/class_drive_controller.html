)
protectedvirtual

曲線走行 ※注意：曲率半径7cm～351cmの範囲内であること。この範囲外を指定したときは一番近い範囲内の値に変更する。

Parameters
pattern走行パターン（曲がる方向を判別する目的）
power基準のパワー値
curvatureRadius曲率半径[cm]
Returns

指定した曲率が許容範囲外であれば、一番近い範囲内の値に変更

パワー値が限界値を超えないようになるまでループ

カーブの曲率半径に適した左右のパワー値を取得

調整後のパワー値が限界値を超えていたら曲率半径を優先するため、パワーと角速度を調整する

超えていたらパワーを下げて再度取得する

調整後のパワー値がマイナスになっていないか確認

超えていたらパワーを上げて再度取得する

変更したパワーをログ出力

実際の回転角度を見ながら左右の出力を調整

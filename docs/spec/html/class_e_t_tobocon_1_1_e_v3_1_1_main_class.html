文字列をBluetooth通信で送る.

Parameters
strString you want to send. (max : 255 bytes)
connectionConnection.

通信プロトコルは次の通りである. ASCIIエンコーディングして送信する.

  • 1byte目 : 送信する文字列長. 1byteで表せる範囲の制限上, 文字列の最大長は255である.
  • 2byte目以降 : 送信文字列を逆順にしたもの. ただし, 逆順処理は当メソッド内で行われるため, str に指定する文字列は元の順でよい.

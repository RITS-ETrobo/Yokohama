EV3との通信確立済みのインスタンスを生成するコンストラクタ

EV3への接続を最大CONNECTION_RETRY_COUNT回試みる. (1回の試行にかかる時間は約1秒) それでも通信確立できなかった場合は, ProtocolProcessor._streamは<c>nullとなる.

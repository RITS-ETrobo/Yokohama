Detailed Description

EV3-PC間通信プロトコルに従って, EV3-PCの通信を行うためのクラス.

シングルトンクラスとして利用する. 使用者はEV3, PCそれぞれで一つずつインスタンスを生成する.

EV3-PC間通信プロトコルについては, docs/design/misc/protocol.mdを参照のこと.
